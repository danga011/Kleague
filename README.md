# K-Scout Adapt-Fit AI : 정량적 전술 적합도 분석 시스템

## 1. 프로젝트 개요
본 프로젝트는 2024년 K리그 경기 데이터를 기반으로, 특정 구단의 전술 스타일에 가장 최적화된 외국인 선수를 추천하는 인공지능 스카우팅 시스템입니다. 

단순히 선수의 개인 기량을 평가하는 기존 방식에서 벗어나, **구단별 고유 플레이스타일**과 **한국 리그 특유의 환경(기후, 문화)**을 종합적으로 분석하여 영입 성공률을 높이는 정량적 지표인 **HSI (Harmonic Synergy Index)**를 제공합니다.

---

## 2. 주요 기능
- **전술 적합도 점수(HSI)**: T-Fit(전술), P-Fit(환경), C-Fit(문화) 지표를 통한 다각도 분석
- **실시간 팀 매칭**: 구단 선택 시 해당 구단의 포지션별 전술 템플릿과 자동 비교
- **AI 심층 스카우팅 보고서**: GPT-4o-mini 기반의 데이터 근거형 전문 리포트 생성
- **PDF 결과서 내보내기**: 레이더 차트와 AI 분석이 포함된 공식 보고서 출력

## 3. 핵심 알고리즘 및 지표

### HSI (Harmonic Synergy Index)
선수와 구단 간의 시너지를 결정하는 세 가지 핵심 축입니다.

#### 1) T-Fit (Tactical Fit)
- **정의**: 전방 압박 강도 및 경기 규율 유지 능력
- **수식**: `T-Fit = T_raw * (0.7 + 0.3 * CleanPlay)`
  - `T_raw`: 상대 진영 내 유효 수비 액션 (Duel, Tackle, Interception)
  - `CleanPlay`: 경기당 파울 및 카드 리스크를 반영한 보정 계수

#### 2) P-Fit (Physical Fit)
- **정의**: 혹서기(6~8월) 활동량 유지 및 체력 기복 관리
- **수식**: `P-Fit = (여름철 경기당 활동량) / (평시 경기당 활동량)`
  - 기후 변화에 민감한 K리그 특성을 반영한 신체 적응도 지표

#### 3) C-Fit (Cultural Fit)
- **정의**: 선수 성장 배경과 구단 연고지 간의 문화적 유사성
- **수식**: WVS(World Values Survey) 도시 벡터 기반의 문화적 거리 산출
  - 성장 도시와 분석 도시 간의 가치관 거리를 계산하여 초기 적응 리스크 수치화

## 4. 실행 및 설치 방법

### 시스템 요구 사항
- Python 3.12+ (Anaconda 환경 권장)
- Streamlit 1.37.1+

### 실행 순서
1. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   ```

2. **환경 변수 설정**
   `.env` 파일을 생성하고 OpenAI API Key를 입력합니다.
   ```
   OPENAI_API_KEY=your_api_key_here
   ```

3. **애플리케이션 실행**
   ```bash
   streamlit run app.py
   ```

## 5. 프로젝트 구조
- `app.py`: 메인 대시보드 애플리케이션
- `src/`: 핵심 알고리즘(HSI, WVS Fit) 및 데이터 처리 엔진
- `data/`: 원본 데이터 및 가공된 도시 문화 벡터 정보
- `output/`: 분석 결과 데이터 및 자동 생성 보고서 저장소
- `assets/`: 구단 로고 및 사업 계획서 등 브랜드 자산
