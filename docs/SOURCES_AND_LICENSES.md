# 외부 데이터/도구 출처 및 라이선스 정리 (제출용)

> 공모전 규정: **외부 데이터/외부 도구/소프트웨어**를 사용한 경우, 각각의 **출처(URL), 데이터셋/논문명, 라이선스**를 제출 PDF에 명시해야 합니다.  
> 이 파일을 채운 뒤, 최종 PDF의 “참고문헌/출처/라이선스” 섹션에 그대로 붙여 넣는 것을 권장합니다.

---

## 1) 데이터셋(필수/외부)

### A. 공모전 제공 데이터(필수 사용)

| 항목 | 데이터셋/파일 | 출처 | 라이선스/비고 |
|---|---|---|---|
| K리그 이벤트 데이터 | `data/raw/raw_data.csv` | (대회 코드 공유 페이지/대회 안내 URL 기입) | 공모전 제공 데이터 규정 준수 |
| 경기 메타 데이터 | `data/raw/match_info.csv` | (대회 코드 공유 페이지/대회 안내 URL 기입) | 공모전 제공 데이터 규정 준수 |

### B. 외부 데이터(선택)

| 항목 | 데이터셋/파일 | 출처 | 라이선스/비고 |
|---|---|---|---|
| WVS Time Series (1981–2022, CSV v5.0) | `data/raw/WVS_Time_Series_1981-2022_csv_v5_0.csv` | World Values Survey (WVS) 공식 사이트/다운로드 페이지 | 재배포/상업사용 조건 확인 필요 |
| WVS 도시 벡터(가공 결과) | `data/processed/wvs_city_vectors.csv` | WVS 원본에서 파생(본 프로젝트 `src/wvs_city_profiler.py`로 생성) | 파생 데이터(derived). 원본 라이선스 준수 |
| 선수 프로필(영문 풀네임/국적 등) | `data/raw/player_profile.csv` | (예: 구단 공식 페이지/리그 공식/검증된 오픈 데이터) | 크롤링 정책/라이선스 확인 필요 |

## 3) 오픈소스 소프트웨어(핵심)
